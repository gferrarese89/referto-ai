Benvenuto in RefertoAI

Questa app genera referti radiologici nel tuo stile a partire dai reperti che scrivi. Per funzionare ha bisogno di una chiave API di Anthropic (Claude).

Come creare la chiave API (2 minuti)
  1. Vai su console.anthropic.com e crea un account.
  2. Aggiungi un metodo di pagamento (si paga a consumo: pochi centesimi per referto).
  3. Consigliato: in Settings → Limits imposta un limite di spesa mensile (es. 20 $).
  4. In API Keys clicca Create Key e copia la chiave (inizia con sk-ant-).

Insegnami il tuo stile

Sincronizzazione tra dispositivi

Sincronizza automaticamente profilo di stile e archivio referti tra Mac, iPhone e PC di lavoro tramite un Gist segreto del tuo account GitHub (gratuito). La chiave API di Claude e il token GitHub non vengono mai sincronizzati.

Come creare il token GitHub (2 minuti)
  1. Vai su github.com/settings/tokens/new (token classic).
  2. Nome a piacere (es. "RefertoAI"), scadenza consigliata: 1 anno.
  3. Seleziona solo lo scope gist — nient'altro.
  4. Clicca Generate token e copia il token (inizia con ghp_).
  5. Incolla lo stesso token su ogni dispositivo: si agganceranno allo stesso profilo.

Portabilità manuale

In alternativa alla sincronizzazione: esporta il profilo (stile, archivio, impostazioni — senza chiavi) su file e importalo su un altro dispositivo.
